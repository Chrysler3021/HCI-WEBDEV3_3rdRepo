Role-Based Access

Built for every
stakeholder.

ZamboCollect supports 4 distinct roles — each with tailored access, KYC verification, and strict hierarchy enforced throughout Zamboanga City.

👤
Citizen
Verified Zamboanga residents
  • KYC verification required with 1 valid government ID + selfie
  • Report trash with photo, location, and description
  • Track report status in real-time
  • View barangay collection map and schedules
  • Profile locked after verification — no unauthorized changes
🏛
Officer
Assigned per barangay
  • Approve/reject KYC for their assigned barangay only
  • Create Driver accounts under their barangay
  • Manage checkpoints and collection routes
  • Review pending reports and dispatch drivers
  • Cannot create Admin or Officer accounts
🚛
Driver
Created by Officer, per barangay
  • View today's optimized route with hotspot navigation
  • Mark stops as collected and upload proof photos
  • Check in at checkpoints along the route
  • View trash density hotspots for priority areas
  • Account created by Officer — cannot self-register
⚙️
Admin
Zamboanga City system control
  • Create Officer accounts for any barangay
  • City-wide analytics across all 98 barangays
  • Monitor all drivers & officers across Zamboanga City
  • Cannot approve KYC or create Driver accounts directly
  • Export reports as PDF or CSV

System-Wide

Powerful infrastructure
under the hood.

Built on a robust, scalable platform with real-time capabilities, KYC verification, and ironclad role-based access control.

🪪
KYC Verification
1 valid government ID + selfie. Applications route automatically to the barangay officer for review.
🔐
Role-Based Access
Strict hierarchy: Admin creates Officers, Officers create Drivers. Citizens self-register with KYC.
🗺
Real-Time Maps
Live map updates show reports, driver positions, and density hotspots across Zamboanga City.
📶
Offline & Auto-Sync
Citizens and drivers can work offline. Queued data syncs automatically when connectivity is restored.
🔔
Push Notifications
KYC status updates, report approvals, and route assignments delivered in real-time.
🔒
Profile Lock
Verified citizen profiles are locked — no edits allowed after KYC approval to prevent fraud.
📊
Analytics & Exports
City-level dashboards for all 98 barangays with exportable PDF and CSV reports.
📋
Audit Trail
Every KYC decision, report, and route completion is logged for full accountability.
